# **App Name**: TripEase

## Core Features:

- Itinerary Generation: Generates a personalized trip itinerary based on user preferences, budget, and travel style, using an AI tool.
- Multi-Agent Orchestration: Orchestrates various specialized agents (FlightFinder, StaySmart, etc.) to handle different aspects of trip planning. The orchestrator agent is a tool which invokes CriticAgent and LoopAgent.
- Itinerary Display: Displays the generated itinerary in a structured and easy-to-read format, including daily plans, total price, and booked items.
- User Profile Management: Allows users to create and manage their profiles, including budget preferences, travel style, and past trips.
- User Feedback and Refinement: Enables users to provide feedback on the generated itinerary, triggering the LoopAgent to regenerate specific parts of the plan.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and reliability.
- Background color: Light gray (#ECEFF1) to provide a clean and neutral backdrop.
- Accent color: Amber (#FFC107) for highlights and call-to-action buttons, providing warmth and energy.
- Body and headline font: 'PT Sans', a humanist sans-serif font providing a modern, legible, and slightly warm feel.
- Use clear, simple icons to represent different aspects of the itinerary, such as flights, accommodations, and activities.
- Design a clean and intuitive layout that prioritizes readability and ease of navigation. Use white space to separate content sections.
- Incorporate subtle animations to enhance user engagement and provide feedback on interactions, such as loading indicators and transitions.
